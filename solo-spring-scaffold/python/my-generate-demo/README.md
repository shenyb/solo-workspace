# my-generate-demo

> Generated by **solo-spring-scaffold**

## 快速开始

```bash
# 前提: JDK 21, Maven 3.8+, IDEA

# 编译全部模块 + 启动 web（H2 内存库，开箱即用）
cd my-generate-demo
mvn spring-boot:run -pl my-generate-demo-web -am
```

访问: http://localhost:8080/swagger-ui.html

## 切换 MySQL

1. 在 `my-generate-demo-dao` 的 pom.xml 中添加 MySQL 驱动（已添加）
2. 修改 `application-prod.yml` 中的数据库连接信息
3. 运行 `bin/gen-tables.sh` 从数据库表生成代码

## 部署到 VPS

1. 编辑 `bin/deploy.sh`，填上服务器地址
2. 运行 `bin/deploy.sh`
